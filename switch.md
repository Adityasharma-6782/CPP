📌 Overview

This project demonstrates the use of switch case statements in C++.
Switch case is used to execute different blocks of code based on the value of a variable. It is commonly used as an alternative to multiple if-else statements for better readability and performance.

🎯 Features
Simple and beginner-friendly code
Menu-driven program
Demonstrates multiple switch case operations
Easy to modify and expand
💻 Program Description

This C++ program performs basic arithmetic operations using switch case:

Addition
Subtraction
Multiplication
Division
Exit

🛠️ Requirements
C++ Compiler (GCC / g++)
Any Code Editor (VS Code, CodeBlocks, Dev C++)

🎓 Learning Outcomes

After completing this project, you will understand:

Switch case syntax
Menu-driven programs
Basic C++ input/output
Program control flow
